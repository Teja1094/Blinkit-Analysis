# Blinkit-Analysis

Project Overview

A Power BI dashboard analyzing Blinkit retail data to track sales, outlet performance, and customer behavior across products and locations.

Objectives

•	Analyze sales performance across products and outlets 
•	Understand customer satisfaction using ratings 
•	Evaluate impact of visibility, weight, and fat content on sales 
•	Provide insights to improve product strategy and outlet performance 

Tools Used

•	Power BI – Dashboard creation and visualization 
•	DAX – Measures and calculated columns 
•	Power Query – Data cleaning and transformation 
•	Excel/CSV – Data source 

Description of Dashboard Pages

Page 1: Sales & Outlet Performance
Shows sales trends, product performance, and outlet-wise analysis using charts and KPIs.
Page 2: Customer & Product Insights
Focuses on customer ratings, product visibility, and their impact on sales using advanced visuals.

Key Insights

•	High visibility products drive more sales 
•	Few item types contribute most of the revenue 
•	Customer ratings moderately influence sales 
•	Outlet size and location impact performance 
•	Low-fat products perform well in specific categories
